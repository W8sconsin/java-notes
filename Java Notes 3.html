Steps and Features of Java Development

Another term for Java APIs is ___________.
- Java Class Libraries.
What are the five phases of Java development?
- Edit, compile, load, verify, execute.
You type Java source code with what kind of program?
- An editor.
What file ending indicates Java source code?
- .java
Vi and TextPad are examples of what kind of program?
- Editors.
What is an IDE?
- An integrated development environment. IDE offer both text editing and debuggers for locating logical errors.
What are logical errors?
- Logical errors are mistakes in the design of program. They may present itself as the code running but resulting in unanticipated behavior. An example might be incorrect operator precedence.
What does the command javac do?
- Javac compiles a Java program.
Running javac in a command window (shell prompt, Terminal, or Command Prompt) creates files with what kind of file extension?
- .class
What do bytecodes represent?
- The tasks to execute in the execution phase.
What does JVM stand for?
- Java Virtual Machine.
What is a virtual machine?
- A virtual machine simulates a computer within another computer. It is also known as a guest while the "real" computer that it operates in is known as a "host".
What does the "java" command do?
- It invokes the JVM and executes a Java application.
What does a class loader do?
- It puts Java classes into primary memory.
What is a bytecode verifier?
- It verifies security of the bytecodes and checks that they are valid.
How do today's JVMs execute code?
- They use a combination of JIT (just-in-time) compilation and slower traditional interpretation.
Define JIT (just-in-time) compilation.
- Hot spots (frequently execute stretches) are identified in the code. These get turned into machine code first, so that these stretches don't need to get reinterpreted. This speeds up execution.
