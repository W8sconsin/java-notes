The Basics of Programming Languages

What are the three general types of programming languages?
- Machine, assembly, and high-level.
Which of the three is Java?
- Java is a high-level language.
What type of language is natively understood by a computer?
- Machine language. The famous ones and zeroes.
What do we mean when we say machine language is machine dependent?
- Machine languages are specific to the type of machine or device that they are designed for. Other language may be machine independent or cross-platform, usable by many systems.
What is an assembler?
- An assembler is a translator program that converts assembly language into machine language.
What do compilers do?
- Compilers convert high-level language into machine language, using an intermediate object that they create. Java and C are compiled languages and use compilers.
What is an interpreter program?
- Slower than compilers, interpreters are used with languages like Ruby, JavaScript, and Python. They translate to machine code without creating an intermediate object, so they are more memory efficient. They are more debugging friendly, too. Compilers stop when they hit their first translation error.
