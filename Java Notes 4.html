Object-Oriented Programming

What is UML?
- Unified Modeling Language is a graphical language for designing software systems to industry standards.
What is an object?
- An object is composed of attributes and behaviors that are related. For instance, a "person" object has attributes — height, weight, IQ, age — and behaviors — sleep(), eat(), saySomething().
Define OOD.
- Object-oriented design is a practice that models software after real-world objects.
In OOD, let's represent a vehicle using class and object relationships. A vehicle is a ________ and its subunits — cars, trucks, wagons, motorcycles — are called ___________.
- Class, objects.
What is inheritance?
- Getting characteristics in a child class from a parent class.
Can objects communicate with one another?
- Yes, they can, via messages.
What is encapsulation?
- Wrapping behaviors (operations) and attributes in objects.
Why is information hiding an important feature of objects?
- It prevents other code from screwing up the inner workings of objects.
What is an interface?
- This defines communications between objects.
How does procedural programming differ from OOP?
- Procedural programming is designed around actions. It is action-oriented. C is a procedural language. The unit of programmin in C is the function.
A Java class implements operations as ______ and attributes as _________.
- Methods, fields.
Instantiation is what?
- Creating objects.
What is a client?
- It is a class that uses another class.
Classes are to objects as blueprints are to _____________.
- Houses.
What are associations?
- Relationships between classes.
What are components?
- Groups of classes that can be reused by developers.
What is pseudocode?
- Pseudocode is informal notation used to plan pieces of program without having to write code yet.
What organization promotes the standardization of object-oriented technologies?
- Object Management Group (OMG).

Web 2.0

When was the first Web 2.0 conference?
- 2004.
What is a "mashup"?
- A mashup combines individual web services from different organizations into a combine resource for developing websites and other technologies.
Define a "web service".
- A web service is a source of data that a website can consume.
What is AJAX?
- Although, it actually predates the Web 2.0 period, it really emerged contemporaneously in around 2005. AJAX allows you to do things in the background of website without having to reload a page.
What is an RSS feed?
- A technology that allows companies to push content to subscribers.
What do RSS and AJAX stand for?
- Really Simple Syndication, and Asynchronous JavaScript And XML.

Software Potpourri

What is Agile Software Development?
- A methodology to rapidly get a minimum viable product out and then iteratively improve upon it rather than depending on traditional waterfall projects.
To refactor code means what?
- Refactoring means rewriting code to make it cleaner without changing existing functionality.
What is a design pattern
- A design pattern is a proven setup for an application architecture.
What is open-source software?
- Open-source technology allows editing of source code by companies and individuals, and the technology is freeish. This opens technologies to more people scrutinizing, and technologies are able to get refined and debugged more quickly.
What operating system is considered by many to be the crowning achievement of open-source?
- Linux.
MySQL is an open-source ______________ used in the ____________ Stack.
- Database Management System, LAMP.
Ruby is a _____________ and Rails is a __________________.
- Scripting language, web application framework.
What is SAAS (Software as a Service)?
- SAAS is a model that allows companies to provide software to companies without the companies having to install and maintain it (as much) locally. The software lives in the provider's servers.
